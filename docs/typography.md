# Typography

## Pairing Strategy
- Primary Serif: long-form editorial authority.
- Primary Sans: data labels, interfaces, metadata.

## Hierarchy
- H1: high contrast serif, tight tracking.
- H2/H3: restrained sans with spacing rhythm.
- Metadata: monospace optional for coordinates/time stamps.

## Rules
- Avoid novelty or futuristic fonts.
- Keep line length 55–75 characters for body text.
