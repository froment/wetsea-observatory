# Guide utilisateur

*Public : visiteur / spectateur. Rien à installer.*

Le pipeline WetSea enrichit chaque vidéo et publie le même fond sur trois surfaces
cohérentes. En tant qu'utilisateur, voici ce que tu trouves et où.

## Sur YouTube (`@wetseatech`)

Chaque vidéo traitée reçoit :
- un **titre** court et précis (≤ 60 caractères) ;
- une **description** qui s'ouvre sur une accroche, puis liste les **chapitres**
  (cliquables — YouTube les détecte automatiquement) et les **sources** ;
- les **chapitres** permettent de sauter directement à une section.

## Sur le site (Hugo)

- `https://next.wetandseaai.fr` (et `https://wst-tech.org`).
- Un **article** par vidéo, généré à partir des mêmes données (titre, accroche,
  faits marquants, chapitres, sources).

## Sur le blog WordPress

- `https://wetandseaai.fr`.
- L'**article « Chronique »** correspondant, avec un rendu riche (mise en forme,
  diagrammes, références).

## Ce que la rigueur te garantit

Chaque fait affiché est étiqueté selon sa nature :
- **[OBSERVED]** — donnée mesurable, vérifiable ;
- **[INFERRED]** — déduction logique ;
- **[HYPOTHETICAL]** — hypothèse, signalée comme non prouvée.

Les sources sont listées explicitement. Pas de sensationnalisme : c'est une
ligne éditoriale assumée.
